# EchoSpace

EchoSpace is a personal memory-mapping app that connects emotions to locations.  
It helps users relive meaningful moments by transforming their location data into reflective stories.

What It Does

EchoSpace shows where you’ve been and what those places meant to you - through memories, songs, photos, and messages.

Why It Matters

Many people move through spaces daily but forget the emotional experiences tied to them.  
EchoSpace helps users reconnect with their own stories, turning movement into memory.

Who It’s For

- Travelers who want to keep emotional records of places they visit  
- Everyday users who love journaling or self-reflection  
- People who use Google Maps or Spotify and want deeper insights into their routines  

Key Features

- Memory Map: View past locations with emotional notes  
- Replay Moments: Relive your memories (music, photos, messages)  
- Smart Reflection: Get gentle insights like “You return here when you need peace”  
- Sync with apps: Optional integrations with Photos and Spotify  

Technologies

- Frontend: React Native (for mobile app)
- Backend: Node.js with Express
- Database: Firebase (for easy sync and user storage)
- Map Integration: Google Maps API
